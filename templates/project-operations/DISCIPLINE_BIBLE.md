## 16. 완료 기준

### Definition of Ready

- [ ] 목적과 사용자 가치가 명확하다.
- [ ] 포함·제외 범위가 있다.
- [ ] 책임 원본·영향 분야·관련 스킬이 정해졌다.
- [ ] 선행 결정·보류 재개·보호 경로가 확인됐다.
- [ ] 완료 기준과 검증 방법이 있다.

### Definition of Done

- [ ] 승인 범위의 결과가 실제 파일에 반영됐다.
- [ ] 자동·수동 검증 결과가 기록됐다.
- [ ] 관련 본책·게이트·스킬·Active Context가 최신이다.
- [ ] 이미지·사운드·데이터 Manifest 영향이 반영됐다.
- [ ] 책임 원본·승인 이미지 변경 시 PDF·Publication Manifest를 갱신했다.
- [ ] 미검증·불일치·남은 위험을 분리했다.
- [ ] 다음 작업과 선행 조건이 기록됐다.
