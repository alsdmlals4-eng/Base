## 9. 완료 기준

- [ ] GPT와 Codex의 최초 읽기 순서가 같다.
- [ ] 질문별 현행 책임 원본이 하나다.
- [ ] L1 이상 작업에 주 책임·영향 분야·변경 유형이 있다.
- [ ] 변경 유형별 필수 갱신 문서가 판정된다.
- [ ] 이미지에 캐노니컬 경로와 상태가 있다.
- [ ] 기존 승인 이미지 임의 교체 금지가 명시돼 있다.
- [ ] PR에서 구조적 누락을 검사하는 경로가 있다.
- [ ] 문서·구현·검증 상태가 분리돼 있다.
- [ ] 새 작업자가 10분 안에 방향·상태·다음 작업·검증을 찾는다.
- [ ] 수행하지 않은 런타임 검증을 완료로 표시하지 않았다.
