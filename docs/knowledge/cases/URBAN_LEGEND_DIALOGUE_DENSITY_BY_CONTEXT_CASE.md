# 사례 — 기능 장면과 관계 장면의 대사 밀도를 분리하기

- 출처 프로젝트: urban-legend / 괴이 기록국
- 상태: 기획 채택·부분 구현 경험
- 주제: 캐릭터 매력을 늘리면서 핵심 조사 흐름을 느리게 만들지 않음

## 1. 문제

캐릭터 대사를 늘리면 인물의 매력과 세계관 전달은 좋아질 수 있지만, 사용자가 위험 상황을 읽고 선택해야 하는 조사 장면에서 긴 대화가 반복되면 다음 문제가 생긴다.

- 현재 질문과 선택이 대화에 묻힘.
- 동료가 사용자의 추론을 대신 설명함.
- 모든 장면의 리듬이 같아짐.
- 중요한 감정 대화가 일반 반응과 구분되지 않음.

## 2. 관찰한 근거

선택 중심 서사 사례에서는 행동 직후의 짧고 관련성 높은 동료 반응이 선택을 기억하게 했다. 반면 일상·후일담처럼 즉시 판단 압박이 없는 장면은 긴 대화와 가치관 충돌을 수용하기 좋았다.

## 3. 채택한 결정

- 조사·위험 장면: 상황지시문, 선택, 짧은 기능 대사 중심.
- 지원 캐릭터 대사: 능력 발동, 위험 경고, 결정적 관찰에 제한.
- 일상·후일담·관계 장면: 행동, 대화, 태도 선택, 후속 반응을 충분히 사용.
- 같은 캐릭터의 판단 기준과 말투는 두 모드에서 유지.
- 설명할 설정은 생활 행동·업무 충돌·책임 차이로 보여줌.

## 4. 제외한 접근

- 모든 조사 지점에서 모든 동료가 한 줄씩 반응.
- 캐릭터 매력을 위해 정답·미확인 정보를 대신 설명.
- 일상 장면을 사건 규칙 재설명으로 사용.
- 대사 길이만 줄이고 장면 역할을 구분하지 않음.

## 5. 검증

- 조사 대사를 가려도 상황과 선택 흐름이 이해되는지 확인.
- 사용자가 동료 반응의 이유를 현재 선택과 연결하는지 확인.
- 일상 장면에서 말투만으로 인물을 구분하는지 확인.
- 조사와 관계 장면의 평균 읽기 시간과 반복 피로 비교.

## 6. 결과

대사를 일괄적으로 줄이거나 늘리는 대신 콘텐츠 모드별 역할과 예산을 분리하는 방향이 채택됐다. 캐릭터성은 대사량보다 타이밍, 판단 기준, 후속 기억에서 더 강하게 드러났다.

## 7. 재사용 가능한 원칙

- 같은 캐릭터라도 사용자가 수행 중인 행동에 따라 대사의 역할과 길이를 다르게 한다.
- 기능 장면은 판단을 돕고, 관계 장면은 의미와 애착을 확장한다.
- 짧은 반응은 선택과 직접 관련될 때 가장 효과적이다.
- 캐릭터 목소리는 어휘보다 무엇을 먼저 관찰하고 어떤 책임을 지는지로 유지한다.

## 8. 적용 시 주의

- 대화 자체가 핵심 조작인 인터랙티브 드라마는 기능·관계 모드가 동시에 작동할 수 있다.
- 음성 연기와 자동 진행이 있는 경우 길이뿐 아니라 청취 시간과 건너뛰기 규칙을 함께 설계한다.
- 모든 일상 장면을 장문으로 만들지 말고 장면의 새 정보와 감정 전환을 제한한다.
