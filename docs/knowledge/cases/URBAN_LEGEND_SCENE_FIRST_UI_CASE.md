# 사례 — 정보 패널 중심 화면을 장면 중심 조사 화면으로 재구성하기

- 출처 프로젝트: urban-legend / 괴이 기록국
- 상태: 구현·화면 검증
- 주제: 많은 기록과 선택 정보를 유지하면서 배경·현재 상황·첫 행동을 먼저 읽게 함

## 1. 문제

조사 화면에 장소, 상태, 단서, 후보, 기록, 메뉴를 한꺼번에 노출하면서 다음 문제가 생겼다.

- 배경과 현장의 비정상 변화보다 패널이 먼저 보임.
- 사용자가 지금 답해야 할 질문을 찾는 데 시간이 걸림.
- 중요한 근거와 과거 기록이 같은 밀도로 표시됨.
- 최소 해상도에서 첫 선택이 화면 아래로 밀림.

## 2. 관찰한 근거

도시 미스터리·관찰형 추리 사례에서는 공간·인물·현재 행동이 한 시선 흐름에 있을 때 장소 자체가 사건으로 기억됐다. 기록 재열람은 중요하지만 현재 판단과 관련된 정보만 가까이 두고, 상세 기록은 필요할 때 여는 편이 탐색 피로가 적었다.

## 3. 채택한 결정

- 한 화면의 중심 질문을 먼저 정의.
- 장소·현재 판단·기록을 역할별 영역으로 분리.
- 현재 판단에 직접 쓰는 근거만 즉시 표시.
- 과거 실패 후보를 지우지 않고 배제 이유와 함께 유지.
- 첫 선택을 최소 해상도 안에 배치.
- 상세 기록은 별도 페이지·오버레이로 열되 현재 장면으로 쉽게 복귀.

## 4. 제외한 접근

- 모든 패널을 단순 삭제.
- 특정 벤치마킹 작품의 화면 프레임과 기록장 외형 복제.
- 중요한 정보를 메뉴 깊숙이 숨김.
- 후보를 자동으로 줄여 사용자의 추론을 대신함.

## 5. 검증

- 1280×720과 1920×1080에서 장소·현재 질문·첫 선택 노출 확인.
- 사용자가 화면을 본 뒤 장소, 이상 변화, 지금 고를 행동을 순서대로 설명하는지 확인.
- 오답·실패 기록을 다시 열어 배제 이유를 확인할 수 있는지 검증.
- 오버레이를 닫은 뒤 포커스와 현재 선택 상태가 유지되는지 확인.

## 6. 결과

현재 질문의 시선 순서가 명확해졌고, 기록량을 줄이지 않고도 장면과 선택이 먼저 읽히는 구조를 만들 수 있었다. 화면 외형 자체보다 역할 분리와 최소 해상도 계약이 재사용 가능한 핵심이었다.

## 7. 재사용 가능한 원칙

- 패널 수를 줄이기 전에 현재 판단에 필요한 정보의 시선 순서를 정한다.
- 기록 재열람과 현재 행동의 거리를 역할에 따라 다르게 둔다.
- 실패 후보는 숨기기보다 배제 이유를 읽을 수 있게 남긴다.
- 목업이 아니라 실제 최소 해상도와 최대 텍스트 길이로 검증한다.

## 8. 적용 시 주의

- 여러 수치를 동시에 비교하는 분석 도구는 장면 우선 구조가 적합하지 않을 수 있다.
- 기록을 숨기는 것이 핵심 장르 규칙이라면 정보 공개 단계 자체를 별도 설계해야 한다.
- 패널을 장면 뒤로 보내더라도 접근 경로와 포커스 복귀는 명확해야 한다.
