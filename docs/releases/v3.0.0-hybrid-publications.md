# Base v3.0.0 — Hybrid Publications

## 결정

- Markdown: 서술형 기획의 기본 책임 원본
- JSON: Registry·Manifest·상태·ID·경로·관계·게임 데이터
- PDF: 책임 원본 변경과 같은 작업에서 항상 동기화하는 사람용 열람본
- DOCX: PDF 생성 임시 파일 또는 선택 출력
- Mermaid: 필요한 문서에서만 사용하는 편집 원본
- Skill Map: `SKILL_REGISTRY.json` 책임 원본, 선택적 자동 생성 Markdown·DOCX, 필수 PDF

## 안전성

- JSON Schema 기반 Registry·구조 데이터·Manifest 검증
- 동일 입력 재실행 시 추적 파일 무변경
- 임시 디렉터리 생성·전 페이지 렌더 검증·Manifest 마지막 원자 교체
- 실패 시 기존 정상 PDF·Manifest·자산 보존
- PDF 최신성과 사람 검수 상태 분리

## 호환성

v2.2.0 자동 호환은 없다. `docs/migrations/v2.2.0-to-v3.0.0.md`의 수동 절차를 따른다.
