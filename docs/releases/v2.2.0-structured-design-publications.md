# v2.2.0 — Structured Design Documents and Human Publications

## 목적

프로젝트 전체와 모든 분야 기획서를 다음 단일 계약으로 통합한다.

```text
AI·자동 검사 → 구조화 JSON 책임 원본
사람 기본 열람 → DOCX·PDF
시각 이해 → 자동 다이어그램·승인 이미지·실제 캡처
최신성 → Publication Manifest의 SHA-256·렌더 검수
```

## 경계

- 운영 라우터 Markdown은 유지할 수 있다.
- DOCX·PDF는 파생본이며 직접 수정해 책임 원본으로 사용하지 않는다.
- 기존 프로젝트는 Audit only → 보존 대조 → 사용자 승인 → 승인 범위 마이그레이션 순서를 따른다.
- 프로젝트 고유 세계관·수치·경로·승인 이미지는 대상 프로젝트에 둔다.
